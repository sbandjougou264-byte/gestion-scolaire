## 📋 À propos du projet
Ce logiciel permet de gérer efficacement les activités scolaires, notamment :
* La gestion des élèves et des classes.
* Le suivi des notes et des résultats.
* L'administration générale de l'établissement.

## 🚀 Installation
Pour installer et tester ce projet en local, suivez ces instructions :
1. Clonez ce dépôt : `git clone https://github.com/sbandjougou264-byte/gestion-scolaire.git`
2. Installez les dépendances : `pip install -r requirements.txt`
3. Lancez le projet : `python gestion.py`
